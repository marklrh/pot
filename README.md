# Pot
**pot** is a parser generator.  

Read more at [Wiki Page](https://github.com/marklrh/p/wiki)

### Dependencies
* MetaOCaml  
  ``opam install 4.02.1+BER``
* Merlin  
  ``opam pin add merlin git://github.com/the-lambda-church/merlin#metaocaml``
* Genlet  
  ``opam install genlet``
